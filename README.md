### 🚀 我的精选项目 | Featured Projects

#### 1. [WikiUp](https://github.com/unendev/WikiUp) - 企业级智能知识库问答平台
`WikiUp` 是一个基于 **RAG (检索增强生成)** 架构的企业级智能知识库问答平台，旨在解决企业内部知识库（如产品手册、技术Wiki等）的智能检索问题。
- **核心功能**:
    - **自然语言交互**: 用户可以通过自然语言对话，从知识库中高效、精准地获取信息。
    - **可追溯性**: 每个AI回答都能追溯到原文来源，确保信息的准确性和可靠性。
    - **优秀的用户体验**: AI的回答采用打字机效果的流式传输，提升了用户体验。
- **技术栈**:
    - **后端**: `Java`, `Spring Boot`, `Spring Security`, `JWT`, `WebSocket`, `DJL (Deep Java Library)`, `PyTorch`, `Redis`, `Docker`
    - **前端**: `Vue 3`, `TypeScript`, `Vite`, `Pinia`, `Vue Router`, `Axios`, `Tailwind CSS`
    - **AI模型**: `paraphrase-multilingual-MiniLM-L12-v2` (文本嵌入), `DeepSeek API` (文本生成)

#### 2. [Personal-Dashboard](https://github.com/unendev/personal-dashboard) - 个人仪表盘
`Personal-Dashboard` 是一个高度可定制的个人信息中心，旨在整合和展示来自不同平台的个人数据和信息流。
- **核心功能**:
    - **多源信息聚合**: 能够从Bilibili、Spotify、YouTube和RSS订阅源等多个平台抓取和展示信息。
    - **个人管理工具**: 集成了待办事项列表(Todo List)、计时器和日志记录等多种个人管理工具。
    - **AI驱动的洞察**: 利用AI技术对聚合的数据进行分析，提供每日摘要和洞察。
    - **动态布局**: 用户可以自由拖拽和调整看板布局，实现高度个性化的信息展示。
- **技术栈**:
    - **前端**: `Next.js`, `React`, `TypeScript`
    - **后端与数据库**: `Prisma`
    - **数据聚合**: 通过API和爬虫脚本从Bilibili、Spotify、YouTube等平台获取数据。
    - **自动化**: 使用Cron jobs实现定时任务，如每日AI摘要生成。

#### 3. [NatureE](https://github.com/unendev/NatureE) - 自然良坊微信小程序
`NatureE` 是一款专注于提供有机和健康产品的电商微信小程序。
- **核心功能**:
    - **电商功能**: 包含商品浏览、分类、购物车、订单管理和用户中心等完整的电商流程。
    - **AI客服**: 集成了AI客服功能，能够回答用户关于产品的咨询。
    - **小程序优化**: 针对微信小程序生态进行了性能优化，确保流畅的用户体验。
- **技术栈**:
    - **前端**: 微信小程序 (`JavaScript`, `WXML`, `WXSS`)
    - **UI框架**: 部分页面使用了 `Vue.js`
    - **组件化开发**: 采用了组件化的开发模式，提高了代码的复用性和可维护性。

#### 4. [zsxqtool](https://github.com/unendev/zsxqtool) - 知识星球内容导出工具
`zsxqtool` 是一个为知识星球用户设计的桌面应用，可以帮助用户抓取和导出知识星球的内容，并以网页的形式在本地进行筛选和排序。
- **核心功能**:
    - **内容抓取**: 能够抓取知识星球中的帖子、评论等内容。
    - **本地查看**: 将抓取的内容生成本地HTML文件，方便用户离线浏览和管理。
    - **筛选与排序**: 提供多种筛选和排序功能，帮助用户快速找到所需信息。
- **技术栈**:
    - **核心逻辑**: `Python`
    - **图形用户界面 (GUI)**: `Qt`
    - **Web前端**: `HTML`, `CSS`, `JavaScript`
